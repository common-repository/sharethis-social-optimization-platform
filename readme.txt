== Description ==
This plugin deploys the code needed to run the ShareThis Social Optimization Platform, a free tool from ShareThis that lets you A/B test your content on Facebook and much more - free!

Learn more here: http://www.sharethis.com/platform/

Signup today, it's free forever: http://platform.sharethis.com/

== Installation ==
Installation is as easy as signing up at http://platform.sharethis.com (it's free!) and then installing this plugin to your website.

To install the plugin:

1. Upload this plugin to your `/plugins` folder
2. Activate the plugin through the Plugins menu in your WordPress dashboard
3. Navigate to Settings->ShareThis Platform and enter your property ID you were provided when signing up at platform.sharethis.com (optional)
